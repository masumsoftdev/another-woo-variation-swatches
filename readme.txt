== Description ==

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/another-variation-swatches-for-woocommerce` directory, or install the plugin through the WordPress Plugins screen.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Make sure **WooCommerce** is installed and activated.
4. Edit a variable product and configure swatches for each attribute.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =
Yes. WooCommerce must be installed and activated.

= Does it work with all themes? =
The plugin follows WooCommerce standards and should work with most modern themes. Minor styling adjustments may be needed depending on the theme.

= Can I translate this plugin? =
Yes. The plugin is fully translation-ready using the `another-woo-variation-swatches` text domain.

= Does it affect performance? =
No. The plugin is lightweight and optimized for performance.
